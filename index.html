Контролюючі запитання

1. Які особливості оголошення змінних у JavaScript (var, let, const)?

var, let, const – це ключові слова для оголошення змінних в JavaScript.

var (старий спосіб): Має функціональну або глобальну область видимості. Дозволяє повторне оголошення та зміну значення. Піднімається (hoisting) і отримує значення undefined.

Має блочну область видимості. Дозволяє зміну значення, але не повторне оголошення. Піднімається, але недоступне до оголошення.

const: Має блочну область видимості. Значення не можна змінити після присвоєння. Обов’язково ініціалізується при оголошенні.

2. Які операції можна виконувати з рядками у JavaScript?

Рядки в JavaScript можуть бути у одинарних (' '), подвійних (" ") або зворотних ( ) лапках.

Одинарні та подвійні лапки однакові, але не підтримують багаторядковий текст. Зворотні лапки дозволяють:

Вставляти вирази через ${...}
Створювати багаторядкові рядки
Використовувати тегові шаблони (func\string``)

1. Конкатенація (об'єднання)
2. Довжина рядка
3. Доступ до символів
4. Зміна регістру
5. Пошук підрядка
6. Вирізання частини рядка
7. Заміна символів
8. Розбиття та об'єднання
9. Видалення пробілів
10. Повторення рядка

3. Яка різниця між == і === при порівнянні значень?

Оператор == порівнює значення без перевірки типів даних, тоді як оператор ===
порівнює значення та типи даних. Ось приклад різниці між операторами == та ===.

4. Як працюють логічні оператори в умовах та які є їхні особливості?

У Java доступні такі логічні операції:
Логічне заперечення , воно NOTабо інверсія. Java позначається символом " !" перед операндом. Застосовується одного операнду.
Логічне і , воно ANDабо кон'юнкція. Позначається символом &між двома операндами, до яких застосовується.
Логічне або Java , воно ж - OR, воно ж - диз'юнкція. Java позначається символом “ |” між двома операндами.
Виключне або , XOR, Сувора диз'юнкція. Java позначається символом “ ^” між двома операндами.
У Java до логічних операторів можна віднести умовне або , що позначається як ||, а також умовне і &&.

5. Як перевірити, чи число є простим?

function isPrime(n) {
if (n < 2) return false; \\ Числа менше 2 не є простими
for (let i = 2; i < n; i++) { \\ Перевіряємо всі числа до n
if (n % i === 0) return false; \\ Якщо ділиться без залишку
}
return true; // Якщо жоден не поділив, тоді просте
}

Перевірка:
console.log(isPrime(2)); // true
console.log(isPrime(10)); // false
console.log(isPrime(7)); // true
